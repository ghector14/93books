Adding a New Book

Old Way

1. Copy HTWFIP.html ‚Üí NewBook.html
2. Copy HTWFIP.css ‚Üí NewBook.css
3. Find/replace all URLs in CSS
4. Update title in HTML
5. Add to index.html:
   <div class="book-7">
     <a href="NewBook.html">
       <div class="book-container">
         <div class="book">
           <img src="bookPics/NB.jpg"/>
         </div>
       </div>
     </a>
   </div>
6. Update CSS grid areas
7. Hope you didn't break anything

~20 minutes per book

New Way

1. Open books-data.js
2. Add entry:
   {
     id: 7,
     title: "New Book",
     author: "Author",
     abbrev: "NB",
     coverImage: "bookPics/NB.jpg",
     coverFront: "URL",
     coverBack: "URL", 
     spine: "URL",
     dateRead: "2024-01",
     genre: "Genre",
     summary: {
       intro: "Intro text",
       sections: []
     }
   }
3. Done!

~2 minutes per book

  • Zero duplication - DRY principle
